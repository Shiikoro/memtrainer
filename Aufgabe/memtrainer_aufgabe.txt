Aufgabe Memory Trainer
======================

Zum Gedächtnistraining soll eine Qt-Anwendung erstellt werden,
bei der Sequenzen von Farben über vier Knöpfe nachgespielt
werden sollen (siehe Screenshots).

Zum einfachen Speichern und Laden solcher Sequenzen sollen
SQLite Datenbankdateien genutzt werden. Sie müssen vom nicht
Programm nicht geändert werden. Stattdessen werden sie ggf.
gelöscht und neu erstellt. Der Aufbau kann der Beispielsequenz
eight.seq entnommen werden.

Anforderungen an die Anwendung:
- Zugriff auf die Datenbankdateien über separate DatabaseManager-Klasse
- Sequenzen müssen geladen und gespeichert werden können
- im Play-Modus soll die Sequenz vorgespielt werden
- im Trainings-Modus muss die Sequenz nachgemacht werden
- Slider und LCD verdeutlichen nur wie weit eine Sequenz durchgespielt ist
- über Start- und Stop-Menüpunkte kann der Ablauf gestartet oder vorzeitig
  beendet werden
- Erstellen neuer (in der Länge einstellbarer) Sequenzen soll über einen
  weiteren Menüpunkt erfolgen können

Option bei eigenem Interesse:
  Wer will, kann den dritten (Editor-) Modus noch implementieren, so dass
  vom Benutzer eigene Sequenzen beliebig zusammengebaut werden können.
